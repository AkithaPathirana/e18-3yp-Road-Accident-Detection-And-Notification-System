TECHNOLOGY STACK

Front-End Technologies

React native is used for mobile development and react is used for web development. They provide rich user interfaces, write custom components, and offers strong community support.

Back-End Technologies

Node is used backend runtime environment. It allows developers to write JavaScript code that runs directly in a computer process itself instead of in a browser.

Aws is the cloud service provider as it allows for reliable, safe cloud solutions. Express js is the backend framework. It provides various features that make web application development fast and easy. MongoDB is the database as it makes it easy for developers to store, manage, and retrieve data when creating applications with most programming languages.
